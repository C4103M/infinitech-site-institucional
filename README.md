# Jr Infinitech - Site Institucional

## Descrição do Projeto

Este é o site institucional da **Jr Infinitech**, uma empresa júnior vinculada ao curso de Tecnologia. Somos formados por estudantes apaixonados por resolver problemas reais através do digital, oferecendo soluções de alto impacto com custos acessíveis.

O site foi desenvolvido utilizando **React**, **TypeScript**, **Vite**, **Tailwind CSS** e outras tecnologias modernas para garantir alta performance e uma experiência excepcional ao usuário.

## Landing Page

A landing page do site institucional é composta pelas seguintes seções, projetadas para apresentar a empresa, seus valores, processos e formas de contato de maneira clara e atrativa.

### Header
- Navegação principal com links para as seções da página.
- Design responsivo e moderno.

### Hero Section
- Título principal: "Transformamos sua presença digital com sites de alta performance e anúncios que vendem."
- Descrição da empresa como "Empresa Júnior de Tecnologia".
- Fundo com gradientes sutis e formas decorativas para um visual impactante.
- Chamada para ação implícita através do design envolvente.

### Value Section (Nossos Valores)
Apresenta os quatro pilares da Jr Infinitech:

1. **Custo Acessível**
   - Preços até 70% menores que o mercado, com a mesma qualidade de grandes agências.

2. **Inovação Constante**
   - Equipe formada por estudantes imersos nas tecnologias mais recentes do mercado.

3. **Atendimento Dedicado**
   - Seu projeto recebe atenção exclusiva com reuniões semanais de acompanhamento.

4. **Respaldo Acadêmico**
   - Supervisionados por professores e apoiados pela universidade e Brasil Júnior.

### Process Section (Nosso Processo)
Descreve o fluxo de trabalho em 4 etapas:

1. **Imersão**
   - Entendemos seu negócio, público-alvo e objetivos para criar uma estratégia sob medida.

2. **Desenvolvimento / Setup**
   - Criamos a solução técnica — seja um site otimizado ou campanhas de tráfego pago estruturadas.

3. **Análise & Otimização**
   - Testamos, medimos e otimizamos continuamente para maximizar seus resultados.

4. **Suporte Contínuo**
   - Não abandonamos o cliente após o deploy. Oferecemos suporte e manutenção contínua.

### About Section (Sobre Nós)
- Título: "Jovens profissionais, resultados reais."
- Descrição da empresa: Vinculada ao curso de Tecnologia, formada por estudantes apaixonados por resolver problemas reais através do digital.
- Foco em soluções de alto impacto com custos acessíveis.
- Atuação em desenvolvimento web moderno e gestão estratégica de tráfego pago.

### Contact Section (Contato)
- Formulário de contato com campos para nome, email, empresa e desafio enfrentado.
- Opções de desafios comuns: "Site lento ou desatualizado", "Poucas vendas online", "Começando do zero".
- Integração com toast para validação de formulário.

### Footer
- Informações de rodapé, incluindo links adicionais e direitos autorais.

### WhatsApp Button
- Botão flutuante para contato direto via WhatsApp, facilitando a comunicação imediata.

## Tecnologias Utilizadas
- **Frontend**: React, TypeScript, Vite
- **Styling**: Tailwind CSS, PostCSS
- **UI Components**: Shadcn/ui
- **Animações**: Framer Motion
- **Ícones**: Lucide React
- **Testes**: Vitest, Playwright
- **Linting**: ESLint

## Como Executar o Projeto
1. Instale as dependências: `bun install`
2. Execute o servidor de desenvolvimento: `bun run dev`
3. Para testes: `bun run test`
4. Para build: `bun run build`
